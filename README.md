Hey there! 

I am a PhD candidate in **theoretical evolutionary biology** at the University of Groningen, in the Netherlands, where I have also been temporarily a lecturer in evolutionary biology and a scientific programmer. 

In my work I make and play with computational models that describe the evolution of species, to help us understand a bit better how biological diversity is generated in the real world.

Enjoy my code!

Cheers, Raph

### Software

* [archgen](https://github.com/rscherrer/archgen): quantitative genetics dataset simulator
* [binprint](https://github.com/rscherrer/binprint): C++ library to save many binary output files
* [readpars](https://github.com/rscherrer/readpars): C++ library to read parameters from text files
* [readsim](https://github.com/rscherrer/readsim): R package to efficiently read simulation data into R
* [setupp](https://github.com/rscherrer/setupp): Cross-platform, modular C++ configuration setup
* [simarray](https://github.com/rscherrer/simarray): Python script to generate many simulation folders

### Research projects

* [balance](https://github.com/rscherrer/balance): :mortar_board: quantifying the imbalance of phylogenies under protracted speciation
* [boplume](https://github.com/rscherrer/boplume): :mortar_board: tempo and mode of plumage color evolution in birds-of-paradise
* [brachypode](https://github.com/rscherrer/brachypode): :mortar_board: model of climate adaptation in a patchy landscape
* [coral-networks](https://github.com/rscherrer/coral-networks): :handshake: analysis of resilience in symbiont communities on Caribbean coral reefs
* [DAISIE2](https://github.com/rscherrer/DAISIE2): :handshake: re-implementing and augmenting a model of phylogenetic island biogeography
* [dewlap](https://github.com/rscherrer/dewlap): :mortar_board: measuring dewlap color differences between habitats in _Anolis sagrei_
* [dominican-lizards](https://github.com/rscherrer/dominican-lizards): :handshake: a study of sexual selection and evolvability in _Anolis_ lizards
* [florida-lizards](https://github.com/rscherrer/florida-lizards): :handshake: a study of urban evolution in _Anolis_ lizards
* [hemigene](https://github.com/rscherrer/hemigene): model to explain the evolution of supergenes and hemizygosity
* [reschoice](https://github.com/rscherrer/reschoice): :mortar_board: model of eco-evolutionary branching under resource choice
* [speciome](https://github.com/rscherrer/speciome): :mortar_board: a model of adaptive speciation with gene networks

:mortar_board: = PhD chapter

:handshake: = collaborative work (not main investigator)
